## 💡 Dicas Valiosas

### 🖼️ Sobre Imagens
- Use JPG para fotos
- Use PNG para gráficos
- Comprima em https://tinypng.com
- Tamanho mínimo recomendado: 150x150px
- Tamanho máximo: 2000x2000px

### 🎨 Sobre Cores
- Não mude a paleta roxo (está perfeita!)
- Se precisar, edite `src/index.css`
- Teste em modo escuro primeiro

### 📱 Sobre Responsividade
- Teste no DevTools (F12)
- Mobile: 375px de largura
- Tablet: 768px de largura
- Desktop: 1920px de largura

- Recomendado: Vercel
- Alternativa: Netlify
- Ambos suportam GitHub
- Deploy automático a cada push
